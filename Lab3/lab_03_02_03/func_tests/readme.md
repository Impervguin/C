# Тесты для лабораторной номер 1

## Входные данные
Натуральный rows, cols
row * col вещественных чисел
натуральная digit

## Выходные данные
Матрица без столбцов, где были числа с digit

## Позитивные тесты
01 - обычный тест
02 - Матрица 10х10
03 - ни одного столбца с цифрой


## Негативные тесты
01 - буквы
02 - размер больше 10
03 - 0 элементов
04 - Цифра больше 10
05 - Цифра меньше 0
06 - все столбцы с цифрой
